ThinkSub2

ThinkSub2는 음성 인식 기반 자막 생성 및 편집 도구로,
오디오 파일에서 자막을 생성하고 편집하며, 
WAV 분할 출력과 다양한 음성 처리 작업을 지원하는 데스크톱 애플리케이션입니다.
Whisper 기반 음성 인식과 자막 편집 기능을 결합하여, 
빠르고 효율적인 자막 작업 환경을 제공하기 위해 제작되었습니다.

[![Watch the video](https://img.youtube.com/vi/gZyli9Te6QY/hqdefault.jpg)](https://youtu.be/gZyli9Te6QY)


주요 기능

🎙 음성 파일에서 자동 자막 생성
📝 실시간 자막 편집
⏱ 단어 단위 타임스탬프 지원
✂ 자막 분할 / 병합 / 삭제
🔊 Whisper 엔진 작업을 위한 Lora 생성
📊 웨이브폼 기반 구간 편집
⚡ Faster-Whisper 엔진 지원
